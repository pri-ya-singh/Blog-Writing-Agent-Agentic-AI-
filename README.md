# 📝 Blog Writing Agent (Agentic AI)

## 📌 Overview
The **Blog Writing Agent** is a multi-agent AI system designed to generate high-quality, structured blog content automatically. It uses **Agentic AI principles** with tools like **LangGraph** and **LangChain** to perform research, planning, and content generation in a coordinated workflow.

This system enhances traditional text generation by integrating **real-time research, improved prompting, and fine-tuned outputs**.

---

## 🚀 Features
- 🤖 Multi-agent blog generation using **LangGraph**
- 🔍 Real-time research integration using **Tavily API**
- 🧠 Advanced prompting & fine-tuned responses
- 📝 Structured blog output (Title, Intro, Body, Conclusion)
- 🎨 Diagram/image generation using **Gemini API**
- ⚡ Streamlit-based interactive UI
- 📤 Export-ready blog content

---

## 🛠️ Tech Stack
- **Language:** Python  

- **Frameworks & Tools:**
  - LangChain  
  - LangGraph  
  - Streamlit  
  - Tavily API (for research)  
  - Gemini API (for diagrams/images)
